Manual Book Kompresaja

Versi 1.0.0
Diperbarui: 24 Juni 2025

1. Pendahuluan

Tentang Kompresaja: Kompresaja adalah aplikasi web yang memudahkan pengguna untuk mengompresi berbagai jenis file langsung dari browser. Dengan desain yang intuitif dan dukungan format yang luas, Kompresaja membantu menghemat ruang penyimpanan dan mempercepat pengiriman file secara online.

Tujuan Manual Book: Dokumen ini disusun untuk memandu pengguna dalam menggunakan aplikasi Kompresaja, mulai dari proses unggah file hingga mengunduh hasil kompresi.

2. Persyaratan Sistem

  • Komputer, laptop, atau perangkat seluler dengan akses internet

Browser yang Didukung:

  • Google Chrome (versi terbaru)
  • Mozilla Firefox
  • Microsoft Edge

3. Fitur Utama

FiturDeskripsi
Kompresi Satu KlikKompres file hanya dengan satu tombol
Dukungan Multi-FormatMendukung dokumen, gambar, PDF, dan lainnya
Antarmuka ResponsifTampilan dapat menyesuaikan dengan berbagai perangkat
Statistik Kompresi Real-timeMenampilkan ukuran sebelum dan sesudah kompresi

4. Panduan Penggunaan

4.1 Akses Aplikasi
  1. Buka browser favorit Anda
  2. Kunjungi: https://kompresaja.web.app (contoh alamat)
4.2 Mengompresi File
  1. Klik tombol "Unggah File"
  2. Pilih file dari perangkat Anda
  3. Setelah file berhasil diunggah, klik "Kompres Sekarang"
  4. Tunggu proses kompresi selesai
  5. Klik tombol "Unduh Hasil" untuk mendapatkan file yang sudah dikompres
4.3 Melihat Statistik Kompresi
  • Ukuran file sebelum dikompresi
  • Ukuran file sesudah dikompresi
  • Persentase penghematan ruang
4.4 Format File yang Didukung
  • Dokumen: .pdf, .docx
  • Gambar: .jpg, .gif .png
  • Lainnya: .txt, .csv

5. Keamanan dan Privasi

  • File yang Anda unggah tidak disimpan di server lebih dari 1 jam.
  • Tidak ada data pribadi yang dikumpulkan saat proses kompresi.
  • Setiap unggahan bersifat terenkripsi.

6. Tips Penggunaan

  • Kompres file dalam jumlah kecil untuk hasil lebih cepat
  • Gunakan koneksi internet stabil
  • Jangan tutup browser saat proses kompresi berlangsung

7. Troubleshooting

MasalahSolusi
File gagal diunggahPastikan ukuran file < 100MB dan format didukung
Proses kompresi lamaPeriksa koneksi internet atau ukuran file
Tidak bisa mengunduh hasilCoba refresh halaman dan ulangi proses
